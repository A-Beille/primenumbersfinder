"Détecteur" de nombres premiers

Ce système permet de trouver tous les nombres premiers jusqu'à 1 000 000 maximum presque instantanément. Pour effectuer cela, il utilise le crible d'Ératosthène, procédé visant à supprimer tous les nombres multiples de nombres eux-mêmes premiers (donc supprimer les multiples de 2, 3, 5, 7, 11 etc...). Ce procédé donnera ainsi tous les nombres premiers.
